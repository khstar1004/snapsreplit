# SnapS - 소셜 미디어 콘텐츠 관리 플랫폼

## 프로젝트 소개
SnapS는 소셜 미디어 콘텐츠를 효율적으로 관리하고 변환할 수 있는 플랫폼입니다.

데모 링크 : https://snaps-jamesen1004.replit.app/


## 기능
- 인스타그램/스레드 계정 연동
- 콘텐츠 자동 변환
- 통계 분석
- 콘텐츠 관리

## 기술 스택
- Python 3.11
- Flask
- SQLAlchemy
- OpenAI
- LangChain
